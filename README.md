# postcard-editor

## Project setup
```
npm i
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm build
```

 - Upload an image
 - Add text overlay via button

3. Version 1 (5 hrs)
 - Zoom in / Zoom out
 - Rotate portrait / landscape

4. Version 2 (3 hrs)
 - Undo / Redo
 - Save edited image
